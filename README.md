## Проект - "Съемная квартира" - REST API service

Содержит информацию
* тарифы за услучи по счетчикам:
    * электро-энергию
    * холодная вода
    * горячая вода
    * водоотведение
* ежемесячные показания счетчиков + скрины:
    * электро-энергию
    * холодная вода
    * горячая вода
* суммы для расчета за месяц
    * электро-энергию
    * холодная вода
    * горячая вода
    * общая сумма за месяц
* выгрузка данных в PDF-отчет
